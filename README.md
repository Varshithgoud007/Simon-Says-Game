# Siman Says Game 
this is a fun memory game
## Technologies used 
-HTML
-CSS
-JAVASCRIPT
## Working
1)press any key on the keyword to start the game 
2)from the above four color buttons a random button blinks
3)if u press the correct blinked color then another color blinks
4)now u need to press the previous blinked color button and current blinked color button
5)in this way the game goes on
6)if you press a wrong color button the GAME END displaying your score
7)and again game starts freshly
